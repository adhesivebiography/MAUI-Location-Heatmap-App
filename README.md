# MAUI Location Heatmap App

This repository contains a .NET MAUI application that tracks user
location, stores GPS data in SQLite, and visualizes location density
as a heatmap on Google Maps.

## Features
- Location tracking with runtime permissions
- SQLite-based local storage
- Google Maps integration
- Heatmap visualization using map overlays

## Technologies Used
- C#
- .NET MAUI
- SQLite
- Google Maps SDK for Android
